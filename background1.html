CSS Position Properties Example

Static Position

This is a static positioned element. It follows the normal flow of the document.

Relative Position

This element has a relative position and is shifted 20px down and 30px to the right from its normal position.

Absolute Position

This element is absolutely positioned with respect to its closest positioned ancestor (or the body if no positioned ancestor exists).

Fixed Position

This element is fixed to the bottom-right corner of the viewport. It stays in the same position even when scrolling.
